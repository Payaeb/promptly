# Screenshots to Take for the Store Listing

Mozilla requires at least 1 screenshot. I recommend taking 3 for a professional listing.
Screenshots should be 1280x800 or similar (just use your normal browser window).

## Screenshot 1: The Popup Menu
1. Load Promptly in Firefox (about:debugging)
2. Navigate to any webpage (a job posting works great for the story)
3. Click the Promptly icon in the toolbar
4. Take a screenshot showing the popup with the buttons visible
   - Windows shortcut: Win + Shift + S, then drag over the popup area

## Screenshot 2: The Settings Page
1. Click "Settings" in the Promptly popup (or go to about:addons, find Promptly, click Preferences)
2. Make sure you have a few buttons visible in the list
3. Take a screenshot of the full settings page

## Screenshot 3: The Button Editor
1. On the settings page, click "Add Button" or edit an existing button
2. Fill in some example fields so it looks complete
3. Take a screenshot showing the editor modal

Save these as PNG files in this folder (store/screenshots/).
Name them: screenshot-1-popup.png, screenshot-2-settings.png, screenshot-3-editor.png
